Addressing Modes

  • Immediate: The operand is specified directly in the instruction itself.
  • Register: The operand is stored in a CPU register.
  • Direct: The operand is the memory address of the data.
  • Indirect: The operand is a memory address, and the actual operand is found at that address.
  • Indexed: The operand is obtained by adding a constant value or the content of another register to a base address.
  • Base-Relative: Similar to indexed addressing, but the base address is stored in a register.
  • Stack: Special addressing mode for operations involving the stack pointer register (SP).
